리액트, sts, h2를 활용해서 게시판 웹앱을 제작한다.
Lombok, Spring Data JPA, H2 Database, Spring Web, Spring Security 의존성을 미리 추가 했다.
db는 tcp 서버 모드로
User엔티티 클래스에서 생성하는 테이블 이름은 @Table 어노테이션을 써서 'List'로 변경한다

Login.js는 로그인 페이지다
아이디입력칸, 비밀번호입력칸, 로그인버튼, 회원가입버튼이 있다.
Login.js에서 회원가입버튼을 누르면 Create.js로 이동.

Create.js는 회원가입 페이지다.
아이디입력칸, 비밀번호입력칸, 비밀번호확인입력칸, 고유번호입력칸, 제출버튼이 있다.
값을 입력한 후 제출 버튼을 누를때
만약 비밀번호와 비밀번호확인의 입력값이 같지않다면 '비밀번호가 일치하지 않습니다.' 알림 출력
만약 이미 List테이블에 입력한 것과 같은 아이디가 존재한다면 '이미 존재하는 아이디 입니다.' 알림 출력.
만약 이미 List테이블에 입력한 것과 같은 고유번호가 존재한다면 '이미 가입되어있습니다' 알림 출력.
만약 List테이블에 입력값과 같은 아이디 또는 고유번호가 없다면 '환영합니다. 로그인 후 이용해주세요' 알림 출력, List테이블에 입력한 데이터 추가, Login.js로 이동.

Login.js에서 아이디와 비밀번호를 입력하고 로그인 버튼을 눌렀을때,
만약 입력한값과 같은 데이터가 List테이블에 존재한다면 Welcome.js로 이동.
만약 List테이블에 존재하지 않는 아이디라면 '등록되지않은 아이디 입니다.' 알림 출력
만약 아이디는 존재하나 비밀번호가 존재하지않는다면 '비밀번호가 맞지않습니다.' 알림 출력

Welcome.js는 메인 게시판 페이지다.
Record테이블에서 글번호 ,제목, 작성날짜, 아이디 칼럼이 출력된다.
제목은 하이퍼 텍스트이며, 클릭하면 Detail.js로 이동
'글쓰기' 버튼이 있다.
'글쓰기' 버튼을 클릭하면, Wright.js로 이동

Wright.js는 글쓰기 페이지다.

Detail.js는 상세내용 페이지다.

'완료' 버튼을 누르면 Record에 글번호, 제목, 내용, 작성날짜, 아이디, 비밀번호가 추가된다.
